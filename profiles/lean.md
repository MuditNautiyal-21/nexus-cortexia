# Lean profile

Maximum token conservation. Use this when budget is tight or the project
is straightforward enough that full discussion overhead isn't justified.

## Overrides

- Skip consensus for all tasks rated "simple" or below
- One-round review only (spec compliance check, skip code quality stage)
- No debate escalation. If review doesn't agree, go with the proposer's approach
- Compress all outputs to 100 tokens or fewer
- No examples in responses. Code only
- Target: 40-60% of standard profile token cost
