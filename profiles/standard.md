# Standard profile (default)

Balanced between token cost and output quality. The default for most work.

## Settings

- Consensus for "moderate" and above
- Quick consensus (propose + review, no debate) for "simple"
- Skip consensus for "trivial"
- Full two-stage review for non-trivial tasks
- Compress dependency outputs to ~200-500 tokens
- Normal output verbosity (concise but clear)
